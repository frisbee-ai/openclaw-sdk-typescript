## Project Overview

- **Project Name**: OpenClaw SDK (TypeScript)

## Commands

```bash
# Install dependencies
npm install

# Build the project
npm run build

# Run tests
npm test

# Run linter
npm run lint

# Format code
npm run format
```

## Architecture

- **src/index.ts**: Main entry point, exports `OpenClawClient` class and `ClientConfig` interface
- **dist/**: Compiled JavaScript output (generated by `npm run build`)

- Follow semantic versioning
