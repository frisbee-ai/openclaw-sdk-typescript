## Context

From `05-CONTEXT.md` decisions:
- **D-01:** Fallback order: v3 → v2 → v1 (newest to oldest)
- **D-02:** Degraded behavior: WARNING log + continue connecting
- **D-03:** Structure: `ProtocolVersionRange[]` array — explicit ordered list
- **D-04:** `negotiate()` tries each range in order when server rejects; logs warning on successful fallback; throws only when all ranges exhausted
- **D-05:** `DEFAULT_PROTOCOL_VERSION` becomes `DEFAULT_PROTOCOL_VERSIONS: ProtocolVersionRange[]`

Integration point: `client-builder.ts:271` has `createProtocolNegotiator({ min: 3, max: 3 })` — must change to array form.

### Task 1: Update ProtocolNegotiator to Accept Version Range Array

<read_first>
- src/connection/protocol.ts
</read_first>

<action>
1. Change `ProtocolVersionRange` interface — no changes needed (already has `min` and `max`)

2. Change `DEFAULT_PROTOCOL_VERSION` constant in `src/connection/protocol.ts` from:
   ```ts
   export const DEFAULT_PROTOCOL_VERSION: ProtocolVersionRange = {
     min: 3,
     max: 3,
   };
   ```
   To:
   ```ts
   export const DEFAULT_PROTOCOL_VERSIONS: ProtocolVersionRange[] = [
     { min: 3, max: 3 },
     { min: 2, max: 2 },
     { min: 1, max: 1 },
   ];
   ```

3. Change `ProtocolNegotiator` constructor signature in `src/connection/protocol.ts` from:
   ```ts
   constructor(range: Partial<ProtocolVersionRange> = {})
   ```
   To:
   ```ts
   constructor(ranges: ProtocolVersionRange[] = DEFAULT_PROTOCOL_VERSIONS)
   ```

4. Change `ProtocolNegotiator` private field from:
   ```ts
   private range: ProtocolVersionRange;
   ```
   To:
   ```ts
   private ranges: ProtocolVersionRange[];
   ```

5. Update constructor body from:
   ```ts
   this.range = { ...DEFAULT_PROTOCOL_VERSION, ...range };
   ```
   To:
   ```ts
   this.ranges = ranges;
   ```

6. Update `getRange()` to return the first range:
   ```ts
   getRange(): ProtocolVersionRange {
     return { ...this.ranges[0] };
   }
   ```

7. Rewrite `negotiate()` method to implement fallback loop:
   ```ts
   negotiate(helloOk: HelloOk): NegotiatedProtocol {
     const serverVersion = helloOk.protocol;

     // Try each range in order
     for (const range of this.ranges) {
       if (serverVersion >= range.min && serverVersion <= range.max) {
         // Found a match
         if (serverVersion < this.ranges[0].min) {
           // Degraded — server version is lower than our preferred
           const logger = console;
           logger.warn(
             `[ProtocolNegotiator] Protocol version ${serverVersion} is lower than preferred ${this.ranges[0].min}, proceeding with degraded mode`
           );
         }
         this.negotiatedVersion = serverVersion;
         return { version: serverVersion, min: range.min, max: range.max };
       }
     }

     // All ranges exhausted — throw with useful error
     const supported = this.ranges.map(r => `${r.min}-${r.max}`).join(', ');
     throw new Error(
       `Protocol version ${serverVersion} is out of all supported ranges [${supported}]`
     );
   }
   ```

8. Update `reset()`:
   ```ts
   reset(): void {
     this.negotiatedVersion = null;
   }
   ```

9. Update `createProtocolNegotiator()` factory signature and body:
   ```ts
   export function createProtocolNegotiator(
     ranges?: ProtocolVersionRange[]
   ): ProtocolNegotiator {
     return new ProtocolNegotiator(ranges);
   }
   ```

10. Update JSDoc for `createProtocolNegotiator()` to reflect new parameter.
</action>
